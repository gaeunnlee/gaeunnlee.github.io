# 01. 인터넷이란?

##  nuts and bolts 관점

*수십억 개의 연결된 컴퓨팅 장비들

- 호스트

  - 종단시스템
  - 컴퓨팅 기능이 있는 시스템

- 통신링크

  - 광섬유 / 구리선 / 라디오 / 위성
  - 전송률 = 대역폭 (bandwidth)
    - 초당 여러 비트로 표현되는, 사용 가능하거나 정보 용적의 비트레이트

- 패킷 교환

  - 패킷(데이터의 덩어리)를 전달함

  - 라우터 (router) 

    - 컴퓨터 네트워크 간에 데이터 패킷을 전송하는 네트워크 장치

  - 스위치 (switch)

    - 연결하는 통신 장비로서 허브보다 전송 속도가 개선된 것
